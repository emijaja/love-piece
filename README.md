# 写真から文章を作成

写真をアップロードして、AI が文章を自動生成するアプリです。

## 機能

- 📸 写真のアップロード
- 🖼️ サンプル画像の選択（すぐに試せます）
- 👁️ 画像プレビュー表示
- ✍️ 文章の自動生成（現在はダミーテキスト）
- 📋 生成された文章のコピー
- 🔄 文章の再生成

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

## 使い方

### 画像アップロードページ (/)

1. 以下のいずれかの方法で画像を選択：
   - 「クリックして画像を選択」エリアをクリックして、ローカルから写真を選択
   - **または**、下に表示されるサンプル画像をクリック（すぐに試せます）
2. 選択した写真がプレビュー表示されます
3. 「文章を生成する」ボタンをクリック
4. 結果ページに自動的に遷移します

### 文章表示ページ (/result)

1. アップロードした画像と生成された文章が並んで表示されます
2. 「文章をコピー」ボタンで文章をクリップボードにコピーできます
3. 「文章を再生成」ボタンで別の文章を生成できます
4. 「新しい画像をアップロード」ボタンでトップページに戻ります
